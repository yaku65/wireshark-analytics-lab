# 5. Fazit

Wireshark ist ein vielseitiges Werkzeug, das tiefen Einblick in den Netzwerkverkehr erlaubt.  
Selbst ohne reale Daten kann man Abläufe, Protokolle und Analyseprozesse dokumentieren.

## Gelernt:
- Grundlagen der Protokolle (ICMP, DNS, HTTP)
- Analyse typischer Netzwerkprobleme
- Strukturierte Dokumentation in Markdown für GitHub
